Números!!!!

Números em JavaScript são "valores de precisão dupla no formato IEEE 754"

Como consequência:

0.1 + 0.2 == ?

Funções parserInt() e parseFloat()

Em geral quando buscamos os valores de alguma elemento HTML, esse valor é retornado como String:

Um código assim retorna strings:

var elemento = document.getElementById('n1').innerHTML

Retornará o valor de n1 que é uma String. Se realizarmos uma adição sobre esse valor, estaremos concatenando:

0.1 + 0.2 == ?

Para obtermos o resultado que desejamos com a adição é necessário usar uma função parserInt() ou parserFloat(), no caso com parserFloat podemos trabalhar com decimais:

document.getElementById('n1').innerHTML
document.getElementById('n2').innerHTML
n1 = Number.parseFloat(n1)
n2 = Number.parseFloat(n2)
var resultado = n1 + n2

0.1 + 0.2 == ?
